# 📊 Marketing Performance Dashboard

## 🚀 Overview

This project is an interactive Marketing Analytics Dashboard built using Power BI. It provides insights into campaign performance, revenue generation, ROI, and customer segmentation.

## 📌 Key Features

* KPI tracking: Revenue, Profit, ROI, Conversions
* Campaign performance analysis
* Subscription tier analysis
* Discount impact on sales
* Interactive filters (Campaign, Product, Subscription Tier)

## 🛠️ Tech Stack

* Power BI
* SQL 
* Excel / CSV dataset
* Python libraires 

## 📈 Insights

* Identified high-performing campaigns based on ROI
* Analyzed revenue contribution by subscription tiers
* Evaluated impact of discount levels on units sold
